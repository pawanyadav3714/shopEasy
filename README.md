🛍️ ShopEasy – Modern Shopping Web App

A fast and responsive shopping web application built using React + Vite.
This project focuses on performance, clean UI, and smooth user experience.

🚀 Features
⚡ Lightning-fast performance using Vite
🎨 Clean and modern UI
📱 Fully responsive (mobile + desktop)
🛒 Shopping-style interface
🔄 Dynamic rendering with React
📦 Optimized production build
